# Optical-wire-winding-for-fpv-v1
Solution for fpv drone pilots who use optical wire to transmit data over long distance to rewind optical wire and reuse it

NOTE: Its my 2025 summer project made for one of my customer. finally decided to release it in public so fpv pilots / hobyist can make similar by some tweeking designs and code according to your taste....

this is version1 of simple idea of how can we rewind optical wire of fpv drones with speed controller and evenly winding mechanism using BO motor and limit switches to changes bo motor direction once reaches to one end

hope this project idea and design helps you to make better versions for community 👍😊
